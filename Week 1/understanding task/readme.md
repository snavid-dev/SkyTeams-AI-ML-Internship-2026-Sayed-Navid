# What is AI?

AI means **Artificial Intelligence**.

It is when we make a computer **smart** — so it can think and learn like a human.

A normal computer only does what you tell it. But an AI computer can **learn by itself** from examples, and then make decisions on its own.

---

## Simple Example

You show a computer 1000 pictures of cats. Now it can look at a new picture and say — *"this is a cat."*

Nobody told it the rules. It **learned** by itself.
